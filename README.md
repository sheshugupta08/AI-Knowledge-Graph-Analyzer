# AI Knowledge Graph & Document Intelligence System

## Overview

This project extracts knowledge from PDF documents using Natural Language Processing (NLP) and visualizes concept relationships using a knowledge graph.

## Features

- PDF text extraction
- Knowledge graph visualization
- Concept frequency analysis
- Word cloud generation
- Concept flow diagram
- Semantic search
- Chat with PDF

## Technologies Used

- Python
- Streamlit
- spaCy
- NetworkX
- PyVis
- Graphviz
- SentenceTransformers
- FAISS

## Installation

Clone the repository:

git clone https://github.com/sheshugupta08/AI-Knowledge-Graph-Analyzer.git

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
